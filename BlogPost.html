O QUE DA PARA FAZER COM CSS?

Bom galera, vocês além de ganharem um ótimo salário, vocês conseguem customizar da forma de vocês:

Paginas, Blogs (como esse), sites e mais um monte de coisas.

De acordo com a MDN o CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo (en-US) usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias. CSS é uma das principais linguagens da open web e é padronizada em navagadores web de acordo com as especificação da W3C. :
